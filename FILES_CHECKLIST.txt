READY TO USE
============

To push wrapper files to GitHub:

  Option 1 - Automated Python:
    python3 final_push.py

  Option 2 - Simple Bash:
    bash SIMPLE_PUSH.sh

  Option 3 - Manual Git:
    git config user.email "your@email.com"
    git config user.name "Your Name"
    git add gradlew gradlew.bat gradle/wrapper/gradle-wrapper.properties
    git commit -m "Add Gradle wrapper files"
    git push origin main

To build locally:

  chmod +x ./gradlew
  ./gradlew assembleDebug --no-daemon

To install on device:

  adb install app/build/outputs/apk/debug/app-debug.apk
