   ./gradlew assembleDebug --no-daemon

2. INSTALL ON DEVICE
   adb install app/build/outputs/apk/debug/app-debug.apk

3. REQUIREMENTS
   - Rooted device/emulator
   - Magisk or SuperSU installed
   - MANAGE_EXTERNAL_STORAGE permission (Android 11+)

4. USAGE
   - Open app
   - Tap "Check Root Access"
   - Enable "Backup App Data" (optional)
   - Enable "Backup Folders" (optional)
   - Select apps to backup
   - Click "Create Backup"
   - Backup saved to app/Backups/

5. RESTORE
   - Click "Restore"
   - Select a backup file
   - Confirm
   - Apps will be reinstalled, data restored
